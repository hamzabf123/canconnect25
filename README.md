# 🇲🇦 CANConnect25

**CANConnect25** is a web application developed for the **Africa Cup of Nations 2025 (CAN 2025)** in Morocco.  
It connects **fans**, **volunteers**, and **organizers** to ensure a smooth, collaborative, and engaging tournament experience.

## 🌍 Purpose

CANConnect25 aims to:

- Facilitate communication between fans, volunteers, and organizers.
- Enable volunteers to register and receive missions from organizers.
- Allow fans to report lost items or request help during events.
- Provide a central platform for organizing and managing CAN 2025 operations efficiently.

## ✨ Key Features

- 📝 Volunteer registration and mission assignment  
- 📣 Real-time notifications from organizers  
- 📍 Help & emergency request system  
- 📦 Lost & found item declaration  
- 🖥️ Modern responsive interface for all users

## 🛠️ Built With

- **React** – Front-end framework  
- **Node.js / Express** *(optional if back-end exists)*  
- **Java / JEE** *(if used in back-end)*  
- **MySQL / Oracle** – Database  
- **Tailwind CSS** – For UI styling
